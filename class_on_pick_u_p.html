Detailed Description

Called when the user picks up any of the spawned in objects like flowers, or potions. These prefabs that are instantiated have a catching speed of 0. This causes the hand to instantly "catch it" after its spawned. After half a second its catching speed is then set to -1 (the default for interactables) to avoid any strange behavior.
