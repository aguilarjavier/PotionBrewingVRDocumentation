\doxysubsection{Detailed Description}
Creates a looping 2D Animation on a Canvas by enabling/disabling the child gameobjects placed in an array.  

Placement of each child gameobject is critical. Animation will start with first item array and then the next. Will then loop once it finishes with the last item.
