\doxysubsection{Detailed Description}
Changes the material\textquotesingle{}s color and emission color values of the gameobject (potion bottle) when left on the stove or ice pot for too long.  

Note\+: Always make sure to keep the Max Emission Value greater than the Emisson Value before starting the game.
